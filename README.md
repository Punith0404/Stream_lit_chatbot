# AI Chatbot (Streamlit + Groq)

## Project Overview

This project is a web-based AI chatbot built using Streamlit and Groq’s language model API.
It enables real-time interaction with an AI assistant and supports multiple response styles such as Expert, Teacher, and Jovial to adapt to different user needs.

---

## Tech Stack

* Python
* Streamlit
* Groq API
* python-dotenv

---

## Problem Statement

The project demonstrates how to build a simple and customizable AI chatbot that can provide instant responses, adapt its communication style, and maintain conversational context.
It serves as a foundation for applications like virtual assistants, learning tools, and support chat systems.

---

## How to Run

### Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Set up environment variables

Create a `.env` file in the root directory and add:

```bash
GROQ_API_KEY=your_api_key_here
```

### Run the application

```bash
streamlit run app.py
```

The application will open in your browser.
